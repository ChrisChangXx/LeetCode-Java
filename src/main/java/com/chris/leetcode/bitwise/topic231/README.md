# [231. 2的幂](https://leetcode-cn.com/problems/power-of-two/)

## 题目描述

给你一个整数，请你判断它是否是 2 的幂次方。如果是，返回 `true` ；否则，返回 `false` 。
如果存在一个整数`x`，使得`2^x=n`，那么就是2的幂次方。

**示例 1:**

```
输入: 1
输出: true
解释: 2^0 = 1
```

**示例 2:**

```
输入: 16
输出: true
解释: 2^4 = 16
```

**示例 3:**

```
输入: 3
输出: false
解释: 2^0 = 1, 2^1 = 2, 2^2 = 4, 2^3 = 8
```

**示例 4:**

```
输入: 5
输出: false
解释: 2^0 = 1, 2^1 = 2, 2^2 = 4, 2^3 = 8, 2^4 = 16
```

**提示:**

- `-2^31 <= n <= 2^31 - 1`

## 解题思路

- 位运算
- n & (n-1) 将最右边的1变为0
